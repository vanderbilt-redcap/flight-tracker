Use Case: Exporting Data

Situation: A biostatistician wants to correlate the receipt of R01 or equivalent grants with publication patterns. She asks for your data in a format that R can access. What do you do?

Solution: There are three items that you might find helpful to share with your statistician.

First, you can give her access to the REDCap project (under User Rights within the project itself), including API Tokens. The tell her to access the REDCap menu’s API Playground. This page will help you generate code to export your records in the R programming language. (There are other languages supported.)

Second, you can download the Data Dictionary from REDCap in the form of a CSV spreadsheet. This can be opened by almost all major spreadsheet programs. This will provide some definitions to the variables.

Finally, you might be interested in sharing the Flight Tracker’s Codebook, which contains definitions of how certain variables are calculated.
