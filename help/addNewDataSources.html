Adding New Data Sources

This process requires a computer programmer proficient in PHP. She/he would need the application to access the new data source on the application’s nightly cron to run an overnight process. There exist some core classes that can help with the data processing (both as a template and as a resource). These are in the module’s classes/ directory.

The following steps would need to be taken:

  1. Access to the data source needs to be established through a PHP script.
  2. Data structures would need to be added to the REDCap form.
  3. The data would need to be transformed into the REDCap data structures and saved in the appropriate records in REDCap. (This requires name matching.)
  4. The software process would need to be tested and debugged.
  5. The software needs to be added to the list of crons specified in small_base.php.

This data source could provide any of the following or something else:

  • Demographic information
  • Information about career advancement (promotions)
  • Grant information
  • Financial information
  • Publication information

This entire process would need to require some give-and-take initially between programmers. As the application matures, this process should be refined into an easier process.
